Основные команды КомандаОписание ls Показать все файлы и каталоги, присутствующие в рабочем каталоге ls -l Показывать все файлы и каталоги вместе с подробностями, такими как владелец, размер и время последнего изменения. ls -t Сортировать вывод lsпо дате последнего изменения pwd Укажите путь к текущему рабочему каталогу cd directory Переключиться на другой рабочий каталог cd .. Перейти на один уровень каталога вверх cd / Перейти в корневой каталог clear Удалить весь текст с экрана терминала history Распечатать все ранее выполненные команды в текущем сеансе touch filename Создать новый файл vi filename Открыть файл для редактирования cat filename Отобразить содержимое файла hostnamectl Вывести список системной информации, включая ядро, выпуск, архитектуру, виртуализацию и т. д. ifconfig Показать сетевые интерфейсы, IP-адреса и MAC-адреса системы date Распечатать текущую системную дату и время top Список всех запущенных процессов в системе free -m Показать статистику использования памяти head filename Отобразить первые 10 строк файла tail filename Отобразить последние 10 строк файла mv file /new/file/path Переместить файл в другое место mv filename new_filename Переименовать файл cp filename new_filename Скопировать файл man command_name Перечислите полезную информацию о команде rm filename Удалить файл rm –rf directory_name Удалить каталог и его содержимое sudo Для обычных пользователей, чтобы запускать команды с повышенными привилегиями mkdir directory_name Создать новый каталог kill pid Уничтожить процесс, используя его идентификатор процесса reboot Перезагрузите систему. shutdown –h now Выключите систему.

Сетевые команды Команда Описание dig domain_name Показать информацию, связанную с DNS для данного доменного имени dig -x host Выполнить обратный поиск хоста host domain_name Выполнить DNS-поиск указанного доменного имени, который выведет IP-адрес. whois domain_name Для получения дополнительной информации о домене ping ip Проверьте соединение между вашим хостом и указанным IP-адресом. ssh username@ip Для безопасного входа в систему под указанным пользователем на другом сервере wget file Загрузить файл wget -c file Продолжить остановленную загрузку traceroute domain_name Отследите маршрут, по которому пакет будет передаваться от вашего компьютера к хосту. telnet domain_name port Подключиться к удаленному хосту через определенный порт netstat –pnltu Отобразить все прослушиваемые в данный момент порты route Отобразите таблицу маршрутизации для вашего компьютера arp Просмотр содержимого таблицы ARP (протокол разрешения адресов) cat /etc/resolv.conf Посмотрите, какие DNS-серверы в данный момент использует ваш компьютер. tcpdump -i eth1 ‘port 80’ Регистрируйте и отслеживайте весь входящий трафик на порту 80. nmap ip Сетевое обнаружение заданного IP. Узнать, работает ли хост, какие порты открыты и т. д.

Команды поиска Команда Описание locate keyword Поиск файла или каталога в предварительно скомпилированном кэше. Гораздо быстрее, чем find, но многие не показывают все результаты find keyword Поиск файла или каталога в реальной системе. Гораздо медленнее, locateно всегда актуален find /home -name *.ext Поиск указанного расширения в /homeкаталоге и всех его подкаталогах. find / -type f ! -perm xxx Список всех файлов, не имеющих указанного разрешения find / -perm /u=r Показать все файлы, доступные только для чтения grep keyword filename Поиск указанного ключевого слова/шаблона в указанном файле grep keyword * Поиск указанного ключевого слова/шаблона во всех файлах в текущем каталоге. grep -i keyword * Игнорировать регистр при поиске grep -r keyword * Выполнить рекурсивный поиск, т.е. включить в поиск подкаталоги. grep -x ‘what to match’ * Распечатать все строки, где найдено совпадение grep -c keyword * Подсчитайте количество совпадений.

Команды разрешений Команда Описание chmod xxx filename Назначить указанные разрешения файлу chmod –R xxx directory Назначить указанные разрешения каталогу и всем его подкаталогам chmod –x filename Удалить разрешения на выполнение файла chown username filename Изменить владельца указанного файла chown username:groupname filename Изменение владельца и группы владельца файла chown username:groupname filename1 filename2 filename3 Изменение владельца и группового владельца нескольких файлов chown –from=bob alice filename Изменить владельца файла можно только в том случае, если он принадлежит определенному пользователю (добавлено после –from=) chown -h usergroup symbolic_link Принудительно изменить владельца и группу символической ссылки

Команды хранения Команда Описание df –h Вывести список информации о хранилище всех разделов, включая общий размер, свободное и используемое пространство и т. д. mount unmount Монтировать или отмонтировать ISO-файл или устройство хранения данных du -h /home/directory_name Найдите размер каталога в удобном для чтения формате du -sh /home/directory_name Чтобы получить общий размер каталога du -ah –exclude=’*.xxx’ /home/directory_name Отображение использования диска всеми файлами в каталоге, за исключением файлов с указанным расширением du -ha –time /home/directory_name Отображение использования диска каталогом по времени изменения fdisk -­l Показывать размер диска вместе с информацией о разделах sudo du -x / | sort -nr | head -20 Перечислите 20 основных каталогов, которые потребляют слишком много ресурсов

Команды управления пользователями Команда Описание adduser username Добавить нового пользователя userdel -r ‘username’ Удалить пользователя passwd -l ‘username’ Изменить пароль пользователя whoami Посмотреть текущего вошедшего в систему пользователя usermod -c ‘This user will be deleted tomorrow’ username Чтобы добавить комментарий к учетной записи пользователя cat /etc/passwd Отобразить список всех пользователей с дополнительной информацией usermod -d /home/test username Изменить домашний каталог пользователя sudo deluser username group_name Удалить пользователя из группы usermod -a -G group_name username Добавить пользователя в группу groupadd group_name Создать новую группу groupdel group_name Удалить группу id Отображение идентификатора пользователя, идентификатора группы и групп для текущего пользователя

Команды для работы с архивами Команда Описание tar -cvf archive.tar filename.txt Сжать файл в архив tar tar -tvf archive.tar Отображение содержимого tar-архива tar -xvf archive.tar filename.txt Извлечь один файл из архива tar zip archive.zip file1.txt file2.html file3.jpg Создать zip-архив с использованием нескольких файлов zip -u archive.zip filename.txt Добавить файл в уже заархивированный файл zip -d archive.zip filename.txt Удалить файл из zip-архива unzip archive.zip Распаковать zip-архив unzip archive.zip -d /directory_name Распаковать zip-архив в определенный каталог tar xf archive Извлечь архив любого типа gzip filename Сжать файл и добавить к нему расширение .gz. Это удалит исходный файл gzip -c filename > archive.gz Создайте новый сжатый файл .gz, сохранив исходный
